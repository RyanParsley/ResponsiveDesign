Typography gotchas

  • You may want to change border widths to reflect font size changes

  • Be careful with your line heights

  • Watch out for awkwardly narrow columns

  • With all these moving parts consistency is more important than ever

  • Navigation links don’t all need to be full width buttons just because you’re using media queries

  • Some of those widows can be prevented

Image gotchas

  • Don’t let them get too big.

    Remember that “too big” means half size in Retna Duplo mode.
